# Project Overview
Digital Literacy equips students with the skills to navigate, evaluate, and create content in a technology-driven world. This project spans five modules covering essential tools — from presentation software and portfolio building to programming platforms, email etiquette, and cybersecurity awareness. Each module is designed to build practical, real-world competence, preparing students to thrive in both academic and professional digital environments with confidence and responsibility.

# Module 1 – Hands-on Training on Presentation Software
This module introduced us to a range of creative tools that go well beyond traditional slides. We explored CoolText for stylish text graphics, Prezi for dynamic non-linear presentations, and Miro for collaborative whiteboarding and brainstorming. Canva turned out to be incredibly versatile — letting us design everything from infographics to social media posts with ease. We also learned how to create and embed GIFs to make presentations more engaging and visually appealing. The hands-on experience with these tools genuinely changed how I think about presenting information — it's not just about bullet points anymore, it's about storytelling through visuals.

# Module 2 – Building a Digital Portfolio
In today's world, your online presence is practically your resume, and this module drove that point home. We set up profiles across platforms like LinkedIn for professional networking, GitHub for showcasing code projects, and Kaggle for data science work. Platforms like ResearchGate and Wiki Education connected us to the academic community, while TechGig, TechEd, and Stack Overflow opened doors to competitions, learning resources, and peer support. Quora helped us practice articulating ideas clearly. Building these profiles felt less like a task and more like laying the foundation of a career — a digital identity that speaks before we do.

# Module 3 – Programming Platforms and Collaborative Learning
This module brought together two important aspects of modern tech education — writing code and working collaboratively. Through Google Workspace tools like Docs, Slides, and Sheets, we practised real-time collaboration and document creation, skills that are essential in any team environment. On the coding side, we got an overview of HackerRank and HackerEarth for competitive programming practice, and CodeChef and CodeTantra for structured challenges. These platforms don't just test your code — they push you to think logically and improve over time. It was a good mix of productivity tools and programming fundamentals that complement each other well.

# Module 4 – Email Etiquette and Social Media Basics
Communication is a skill often taken for granted, and this module made us reconsider that. We studied the importance of email etiquette — how the tone, structure, and clarity of an email can shape the way others perceive you professionally. From subject lines to sign-offs, every element matters. Workplace email etiquette took this further, covering how to handle sensitive situations, maintain professionalism under pressure, and keep communication respectful and concise. Social media etiquette was also discussed — how what we post publicly reflects on us as professionals. It was a surprisingly practical module, reminding us that soft skills carry just as much weight as technical ones.

# Module 5 – Awareness on Cybercrime and Financial Fraud
This module was perhaps the most eye-opening of all. We explored the different types of cybercrime — from phishing and identity theft to more targeted personal and financial attacks. Real-world examples helped illustrate how quickly someone can fall victim if they're not careful. The cybercrime flowchart was particularly useful in understanding how these crimes unfold step by step and where intervention is possible. We also discussed prevention measures — strong passwords, two-factor authentication, verifying sources before clicking links, and what to do if you're targeted. In an era where digital transactions are the norm, this awareness isn't optional — it's essential.
